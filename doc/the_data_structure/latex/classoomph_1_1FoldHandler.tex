\doxysection{oomph\+::Fold\+Handler Class Reference}
\hypertarget{classoomph_1_1FoldHandler}{}\label{classoomph_1_1FoldHandler}\index{oomph::FoldHandler@{oomph::FoldHandler}}


A class that is used to assemble the augmented system that defines a fold (saddle-\/node) or limit point. The "{}standard"{} problem must be a function of a global paramter $\lambda$, and a solution is $R(u,\lambda) = 0 $ , where $ u $ are the unknowns in the problem. A limit point is formally specified by the augmented system of size $ 2N+1 $.

\doxysubsection{Detailed Description}
A class that is used to assemble the augmented system that defines a fold (saddle-\/node) or limit point. The "{}standard"{} problem must be a function of a global paramter $\lambda$, and a solution is $R(u,\lambda) = 0 $ , where $ u $ are the unknowns in the problem. A limit point is formally specified by the augmented system of size $ 2N+1 $. 

\[ R(u,\lambda) = 0, \] \[ Jy = 0, \] \[\phi\cdot y = 1. \] In the above $ J $ is the usual Jacobian matrix, $ dR/du $, and $ \phi $ is a constant vector that is chosen to ensure that the null vector, $ y $, is not trivial.
