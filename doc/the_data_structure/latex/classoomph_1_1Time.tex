\doxysection{oomph\+::Time Class Reference}
\hypertarget{classoomph_1_1Time}{}\label{classoomph_1_1Time}\index{oomph::Time@{oomph::Time}}


Class to keep track of discrete/continous time. It is essential to have a single \doxylink{classoomph_1_1Time}{Time} object when using multiple time-\/stepping schemes; e.\+g., in fluid-\/structure interaction problems, it is common to use different schemes for the fluid and solid domains. Storage is allocated for the current value of the (continuous) time and a limited history of previous timesteps. The number of previous timesteps must be equal to the number required by the "{}highest order"{} scheme.
