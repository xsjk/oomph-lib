\doxysection{oomph\+::Dense\+LU Class Reference}

Dense LU decomposition-\/based solve of full assembled linear system. VERY inefficient but useful to illustrate the principle. Only suitable for use with Serial matrices and vectors. This solver will only work with non-\/distributed matrices and vectors (note\+: \doxylink{classoomph_1_1DenseDoubleMatrix}{Dense\+Double\+Matrix} is not distributable)
