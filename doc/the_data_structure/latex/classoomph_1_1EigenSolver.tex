Solve the real eigenproblem that is assembled by elements in a mesh in a \doxylink{classoomph_1_1Problem}{Problem} object. Note that the assembled matrices include the shift and are real. The eigenvalues and eigenvectors are, in general, complex, and the eigenvalues can be Na\+Ns or Infs. This function is therefore merely provided as a convenience, to be used if the user is confident that Na\+Ns don\textquotesingle{}t arise (e.\+g. in Arnoldi based solvers where typically only a small number of (finite) eigenvalues are computed), or if the users is happy to deal with Na\+Ns in the subsequent post-\/processing. Function is virtual so it can be overloaded for Arnoldi type solvers that compute the (finite) eigenvalues directly At least n\+\_\+eval eigenvalues are computed.
