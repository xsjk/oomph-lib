\doxysection{oomph\+::Trilinos\+Aztec\+OOSolver Class Reference}
\hypertarget{classoomph_1_1TrilinosAztecOOSolver}{}\label{classoomph_1_1TrilinosAztecOOSolver}\index{oomph::TrilinosAztecOOSolver@{oomph::TrilinosAztecOOSolver}}


An interface to the Trilinos Aztec\+OO classes allowing it to be used as an Oomph-\/lib \doxylink{classoomph_1_1LinearSolver}{Linear\+Solver}. The Aztec\+OO solver is a Krylov Subspace solver; the solver type (either \doxylink{classoomph_1_1CG}{CG}, \doxylink{classoomph_1_1GMRES}{GMRES} or \doxylink{classoomph_1_1BiCGStab}{Bi\+CGStab}) can be set using \doxylink{classoomph_1_1TrilinosAztecOOSolver_a7c81cd1f21720d2c05db9d499dc8f2bc}{solver\+\_\+type()}. This solver can be preconditioned with Trilinos Preconditioners (derived from \doxylink{classoomph_1_1TrilinosPreconditionerBase}{Trilinos\+Preconditioner\+Base}) or Oomph-\/lib preconditioners (derived from \doxylink{classoomph_1_1Preconditioner}{Preconditioner}). Preconditioners are set using \doxylink{classoomph_1_1IterativeLinearSolver_ad4ae617ce8304f17bd42f4dc39813e22}{preconditioner\+\_\+pt()}.

\doxysubsection{Detailed Description}
An interface to the Trilinos Aztec\+OO classes allowing it to be used as an Oomph-\/lib \doxylink{classoomph_1_1LinearSolver}{Linear\+Solver}. The Aztec\+OO solver is a Krylov Subspace solver; the solver type (either \doxylink{classoomph_1_1CG}{CG}, \doxylink{classoomph_1_1GMRES}{GMRES} or \doxylink{classoomph_1_1BiCGStab}{Bi\+CGStab}) can be set using \doxylink{classoomph_1_1TrilinosAztecOOSolver_a7c81cd1f21720d2c05db9d499dc8f2bc}{solver\+\_\+type()}. This solver can be preconditioned with Trilinos Preconditioners (derived from \doxylink{classoomph_1_1TrilinosPreconditionerBase}{Trilinos\+Preconditioner\+Base}) or Oomph-\/lib preconditioners (derived from \doxylink{classoomph_1_1Preconditioner}{Preconditioner}). Preconditioners are set using \doxylink{classoomph_1_1IterativeLinearSolver_ad4ae617ce8304f17bd42f4dc39813e22}{preconditioner\+\_\+pt()}.
