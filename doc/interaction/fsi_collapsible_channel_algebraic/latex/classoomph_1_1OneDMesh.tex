\[ x \in [Xmin,Xmax] \] The mesh has two boundaries\+:
\begin{DoxyItemize}
\item Boundary 0 is at $x=Xmin$.
\item Boundary 1 is at $x=Xmax$.
\end{DoxyItemize}There is one node on each of these boundaries.
