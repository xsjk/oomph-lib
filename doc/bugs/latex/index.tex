\chapter{Bugs and known problems}
\hypertarget{index}{}\label{index}\index{Bugs and known problems@{Bugs and known problems}}
The library is under active development and we greatly appreciate \href{../../contact/html/index.html}{\texttt{ any feedback from users}}. If you think you have discovered a bug, or have suggestions for improvement and other constructive criticism please contact the developers. Feel free to get in touch with us directly via

\begin{center} {\ttfamily  oomph-\/lib AT maths DOT man DOT ac DOT uk } \end{center} 

or (preferably) use our Git\+Hub-\/based bug tracking system, accessible online at ~\newline
~\newline
\begin{center} \href{https://github.com/oomph-lib/oomph-lib/issues}{\texttt{ https\+://github.\+com/oomph-\/lib/oomph-\/lib/issues}} \end{center} 

where you will also find a list of bugs we are currently aware of, as well as bugs that have already been resolved since the release of the latest "{}official"{} version.

However, before reporting any bugs, {\bfseries{please}} make sure you study the relevant documentation and the \href{../../FAQ/html/index.html}{\texttt{ FAQs}}, which contain specific instructions on how to report bugs.
